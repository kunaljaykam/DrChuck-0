# Dr Chuck

Just change the src/DrChuck.js to make any update to the website. Customize theme by using your own color scheme globally in the src/\_globalColor.scss file.

## Installation

To get started.

```bash
$ npm install
$ npm start
```

## Usage

Create .env file
Add Your Github Token & Your Github Username
